Download instructions

  • Go to snaptube.in, click on Download
  • If your browser pops up a windows, click on OK; if there’s no popup, the download will start automatically
  • When the download is finished, you can access it from the notification bar, click it to install
  • If you’re seeing this page (install blocked), please go to Settings - Security, and turn on “Unknown sources”
